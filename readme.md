# Enjoy Trip

## Project
* 2023년 SSAFY 관통 프로젝트 아기돼지삼남매 팀
* 여행지 정보 조회 및 여행 계획 웹
* API - https://github.com/youngseo9603/food_trip/wiki

## 참여 인원
|이름|역할|
|------|---|
|박태양|웹 퍼브리셔|
|좌소연|프론트엔드 개발 및 데이터베이스 설계|
|이영서|백엔드 개발|
